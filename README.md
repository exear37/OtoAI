# OtoAI - AI-Powered Automotive Diagnostics & Support System

OtoAI is a dynamic web application that allows users to enter vehicle symptoms and receive AI-driven fault analysis, potential solutions, and maintenance recommendations. This project was developed as a university graduation project.

## 🚀 Features
* **AI Integration:** Real-time and smart diagnostic reports generated based on user complaints using Google Gemini API.
* **User-Friendly Interface:** Fully responsive, modern, and clean web design.
* **Fast & Dynamic Architecture:** Optimized backend and frontend structures built with PHP and modern web technologies.

## 🛠 Tech Stack
* **Backend:** PHP
* **Frontend:** HTML5, CSS3, Bootstrap / Tailwind CSS, JavaScript
* **API:** Google Gemini API

## ⚠️ Installation & Setup
1. Clone or download this repository.
2. Add your own **Gemini API Key** inside the code (where the API request is made):
   `$api_key = "YOUR_GEMINI_API_KEY";`
3. Run the project on a local server environment (XAMPP, WampServer, etc.).
